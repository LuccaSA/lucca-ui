Important: The js part of lucca-ui is an angular framework so is obviously dependent on angular. It also uses moment.js for its manipulation of dates as moment.js is super awesome. It also uses underscore.js, which is a Javascript library providing very useful helpers (especially for arrays and collections).
The luid-user-picker directive depends on AngularJS ui-select because it offers a large set of customizable features.

The exact list of dependencies which can be found in the bower.json file is:

  • lucca-icons {{ dependencies.icons }}
  • angular {{ dependencies.angular }}
  • angular ui bootstrap {{ dependencies.nguibs }}
  • momentjs {{ dependencies.moment }}
  • underscorejs {{ dependencies.underscore }}
  • ui-select {{ dependencies.uiselect }}
  • angular translate {{ dependencies.translate }}
  • angular notify {{ dependencies.notify }}
  • ui-cropper {{ dependencies.crop }}
