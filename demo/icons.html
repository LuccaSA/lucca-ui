Important: all these icons come from the bower componenet lucca-icons, you are not forced to use this library of icons and lucca-ui can survive without.

The icon font is by default fetched from cdn.lucca.fr. You can change it by following the steps available in the readme of lucca-icons.

How to use

Just append/prepend a <i class="lui icon {icon-name}"></i> element.

Important: the words order is important in the icon-name : lui icon arrow south is not the same that lui icon south arrow

Finally, you can use the available mixin .lui_icon_maker(forbidden)
