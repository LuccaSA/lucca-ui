DOCUMENTATION - Getting up and running with the framework

Important: The js part of lucca-ui is an angular framework so is obviously dependent on angular. It also uses momentjs for its manipulation of dates cuz momentjs is super awesome

The exact list of dependencies which can be found in the bower.json file is:

  • angular 1.4.5
  • angular ui bootstrap 0.13.4
  • momentjs 2.10.6

None of the filters displayed here use angular-bootstrap

luifPlaceholder

A basic filter that display a placeholder if the value is undefined or ""

usage: 

{{ myValue | luifPlaceholder:"a fixed placeholder" }}
{{ myValue | luifPlaceholder: aVarInScope }}

myValue:

dynamic placeholder:

myValue: {{myValue}}

with fixed luifPlaceholder: {{ myValue | luifPlaceholder:"a fixed placeholder" }}

with dynamic luifPlaceholder: {{ myValue | luifPlaceholder:myPlaceholder }}

luifDefaultCode

A basic filter that capitalizes and replace spaces by _

usage: 

{{ myValue | luifDefalutCode }}

value:

myValue: {{myValue}}

with luifDefaultCode: {{ myValue | luifDefaultCode }}

luifStartFrom

A filter for ng-repeat if you want to start from a certain index and don't wat to use ng-if="$index >= myIndex"

usage: 

<ANY ng-repeat="val in values | luifStartFrom : 2">...</ANY>
<ANY ng-repeat="val in values | luifStartFrom : myIndex">...</ANY>

myIndex:

all the values ng-repeat="val in values"

  • $index: {{$index}} - value: {{val}}

all the values starting from myIndex ng-repeat="val in values | luifStartFrom : myIndex"

  • $index: {{$index}} - value: {{val}}

luifMoment

A filter to display a date, a moment or a string using the formats provided by momentjs

Will try to transform the input to a moment, if the resulting moment is not valid it will just return the input

if none is provided, will use the format LLL

usage: 

{{ myValue | luifMoment : "LL" }}
{{myValue | luifMoment : myFormat }}

myValue: (enter '2015-01-01' or such like, or try something that is not parsable by moment)

set myValue to

new Date()
moment().startOf('week')

myFormat:

myValue formatted: {{ myValue | luifMoment : myFormat }}
