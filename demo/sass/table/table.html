Tables

To apply Lucca-UI default style for tables, just add the .lui.table classes to your table.
Addind a .{color} class to a row will display a colored left bar on this row.
Lucca-UI also provide a full table directive with advanced features (such as fixed and scrollable columns, filtering and order, etc.) : luid-table-grid.
