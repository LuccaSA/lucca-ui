LabelsLabels allow to display more information in a colorful way. There are no action linked to those element.

Base syntax

To display a default label (unstyled), just add the classes .lui.label to your element. It'll look like this: Default Then you can add more classes to style your label:

Color classes

  • .primaryPrimary
  • .secondarySecondary
  • .yellowYellow (Lucca's color)
  • .greyGrey
  • .lightLight
  • .redRed
  • .orangeOrange
  • .greenGrey

Size classes

  • .x-smallx-small
  • .smallsmall
  • .largelarge
  • .x-largex-large

Addon support

  • .lui.with.addon.greenAddonWith addon
