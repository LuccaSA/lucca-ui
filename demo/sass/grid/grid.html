lui container classes. Columns must be given the lui column classes.

To define the size of a grid element for different devices, x being the number of columns, use

  • lui column mobile-x for mobiles phones
  • lui column tablet-x for tablets
  • lui column desktop-x for desktops
  • lui column large-x for large screens

Use the {device}-offset-x class to offset a specific grid child.

Use the {device}-push-x and {device}-pull-x classes to reorder children elements.

Don't want any margin on the sides of your container ? Give it the fitting class.
