The gridGrid helps organizing the page.

Base syntax

By default, the lucca grid is 12 columns based. This can be overriden.
Grid is made of one (or more) container(s) defined by the .lui.container classes, and columns, set by the .lui.column classes.

Options for container and columns

Container with .fitting class added, to fit 100% of the available width:

 

To define the size of a grid element for different devices, x being the number of columns, use:

  • .lui.column.mobile-x for mobiles phones
  • .lui.column.tablet-x for tablets
  • .lui.column.desktop-x for desktops
  • .lui.column.large-x for large screens
