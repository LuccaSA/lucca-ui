The grid

By default, the grid is 12 columns based, but that can be overriden.

Container should have the lui container classes. Columns must be given the lui column classes.

To define the size of a grid element for mobiles, give it the lui column mobile-x classes where x is the number of columns. For tablets, use lui column tablet-x, for desktop, lui column desktop-x and finally, for large screens, lui column large-x.

You can offset a specific grid child with the {device}-offset-x class.

You can reorder children elements with the {device}-push-x and {device}-pull-x classes.

Example #2: With gutter space

Adding the spaced class to a column will give it a gutter. Adding it to the .lui.container parent element will give gutters to all .lui.column children.

To neutralise gutter space around the container, wrap .lui.column elements in a .lui.columns element (plural).
