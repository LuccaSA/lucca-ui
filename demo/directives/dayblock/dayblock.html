luid-day-block

An element directive to to draw a day block calendar, like this:

or

Features

  • Supports color class adjectives like primary or green
  • Supports size class adjectives like small or x-large

Supported arguments

  • date : The date you want to display, see luifMoment for supported values.
  • show-day : this argument must be a boolean, if true the directive shows the name of the day on top.
  • primary-color : use these two arguments to have your own color palette.
  • secondary-color : see above

Usage

<!-- basic usage -->
<luid-day-block date="myDate"></luid-day-block>
<!-- with a palette and size -->
<luid-day-block date="myDate" class="lui primary large"></luid-day-block>
<!-- with custom colors -->
<luid-day-block date="myDate" show-day="showDay" primary-color="customPrimaryColor" secondary-color="'white'"></luid-day-block>
