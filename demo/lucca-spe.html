luid-user-picker

An input to have a ui-select plugged to /api/v3/users with a more powerful search and also manages former employees and homonyms.

Features:

Supports up and down key events and scroll event to increment/decrement the value the value.

Supported arguments:

  • ng-model: the variable to bind to
  • show-former-employees: boolean indicating if you want to fetch former employees

Usage:

<luid-user-picker ng-model="myUser"></luid-user-picker>
