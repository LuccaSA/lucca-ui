Static animations

Base syntax

To animate an element, add the .lui.animated classes.

Animation type

There are 2 kinds of static animation :

  • .bounce
    Click to bounce
  • .pulse
    Click to pulse

Animation options

By default, animation occur juste once. If you want it to run indefinitely, add the .infinitely class.
