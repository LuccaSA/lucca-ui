DOCUMENTATION - Getting up and running with the framework

Important: The js part of lucca-ui is an angular framework so is obviously dependent on angular. It also uses momentjs for its manipulation of dates cuz momentjs is super awesome

The exact list of dependencies which can be found in the bower.json file is:

  • angular 1.4.5
  • angular ui bootstrap 0.13.4
  • momentjs 2.10.6

None of the filters displayed here use angular-bootstrap
