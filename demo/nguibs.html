Lucca UI

Code on github

DOCUMENTATION - Getting up and running with the framework

Important: The js part of lucca-ui is an override of the styles of directives from angular ui bootstrap to be more compliant with the general aspect of lucca-ui.
For now this part only properly functions when the bootstrap css is present as well in your page. Developments are underway to remove this contraint.

The exact list of dependencies, which can be found in the bower.json file, is:

  • lucca-icons 1.1
  • angular 1.4
  • angular ui bootstrap 0.13
  • momentjs 2.10 - this part does not use this dependency

Datepicker

github.io page

Inline

In a popup

Usage:

<!-- inline -- >
<div class="lui nguibs-datepicker">
	<datepicker ng-model="dt"></date-picker>
</div>
<!-- popup -- >
<div class="lui nguibs-datepicker long input">
	<input type="text" datepicker-popup="yyyy-MM-dd" ng-model="dt" is-open="status.opened" close-text="Close" ng-click="status.opened=true"/>
</div>

Remarks:

There is some problem with the inline component, an icon appears in the week number. There is an issue open for it
