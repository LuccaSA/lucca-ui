Getting started

Dependencies

Lucca-ui does not depend from bootstrap or normalize but it is highly recommended to use them. It was developped for application using angular-ui-bootstrap meaning the bootstrap dependency was always there

This page references both bootstrap.css and normalize.css

Installation

bower install lucca-ui
