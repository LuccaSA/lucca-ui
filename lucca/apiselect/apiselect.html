luid-api-select

an easy to use ui-select plugged to one of the many lucca's apis

implements a basic paging, it will load more results when you hit the bottom of the scroll

Supported arguments

  • ng-model: will have fields id and name
  • api: the api you wnat to hit
  • filter: a filter you want applied to said api
  • order-by: the order you want applied to the result, must look like name,asc or createdAt,desc
