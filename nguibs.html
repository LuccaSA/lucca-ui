Angular ui bootstrap provides a number of directives. Lucca-ui supports only part of it, some of them natively and some need a special treatment to avoid lucca-ui's style drooling everywhere on the page

Important note: we are currently supporting the version 0.13.x of ui-bootstrap and in the version 0.14.x they added the prefix uib- on a lot of their directives (see the change log for more info). For now Lucca-ui does not support the version 0.14.x but it will be added at some point, see issue 39.

Here is the list of supported angular ui bootstrap directives:

  • buttons
  • collapse
  • datepicker - see below for how to implement
  • dropdown - see below for how to implement
  • modal - there are some shenanigans at work here when you have a namespanced distribution of lucca-ui, issue 46 addresses this kerfuffle
  • popover - see below for how to implement
  • tooltip - see below for how to implement
  • ui select - not part of the angular ui bootstrap suit per se but treated as such for convenience - see below for implementation details

Dropdown

github.io page

  • action
  • disabled action
  • another action

Usage:

<div class="lui nguibs-dropdown input" dropdown dropdown-toggle>
	<button class="lui button" dropdow-toggle>click me</button>
	<ul class="dropdown-menu">
		<li><a>action</a></li>
		<li class="disabled"><a>disabled</a></li>
	</ul>
</div>
